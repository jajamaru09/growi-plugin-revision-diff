# 01-requirements.md

## 変更・追加する機能の説明

HTMLモードの差分表示において、Growi の実際のレンダリングエンジンを使用して各リビジョンを
HTML化し、差分をピンク・緑でハイライト表示する。

具体的には、各リビジョンを Growi の通常ページとして隠し iframe に読み込み、
レンダリング完了後にコンテンツ HTML を取得して差分算出に使用する。

同一オリジン上で動作するためiframeのDOMアクセスが可能。

## ユーザーストーリー

- ユーザーとして、HTMLモードで差分を確認する際に Growi で記事を閲覧するのと同じ見た目で
  差分（ピンク・緑のハイライト）を確認したい

## 受け入れ条件

- HTMLモードで2つのリビジョンを選択すると、Growi スタイルでレンダリングされた差分が表示される
- 追加部分は淡い緑、削除部分は淡いピンクでハイライトされる
- Markdownモードの動作は変わらない
- iframe読み込み中はローディング表示が出る

## 制約事項

- 各リビジョン比較に隠し iframe を2つ使用するため、初回表示は数秒かかることを許容する
- `growiFacade` を使った HTML レンダリングは削除する（iframe 方式に完全移行）
- ページパスは `window.location.pathname` または API から取得する
