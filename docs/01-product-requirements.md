# プロダクト要求定義書

## プロダクトビジョンと目的

Growi Wiki上で閲覧中の記事について、任意の2つのリビジョン間の差分（diff）を確認できるプラグインを提供する。
Markdownテキスト形式とHTMLレンダリング形式の両方での差分確認を可能にし、記事の変更履歴を視覚的に把握しやすくする。

## ターゲットユーザーと課題・ニーズ

### ターゲットユーザー
- Growi Wikiを利用して文書管理・知識共有を行うチーム・組織のメンバー
- 記事の変更履歴を追跡・レビューしたい編集者・管理者

### 課題・ニーズ
- 記事がいつ・どのように変更されたかを素早く確認したい
- Markdown raw テキストと実際のHTMLレンダリング結果の両面から差分を確認したい
- Growi標準のUI体験を損なわずに差分確認機能を利用したい

## 主要な機能一覧

1. **差分確認モーダルを開くボタン**
   - Sidebarに配置
   - Growi標準のSidebarボタンと同じHTML/CSSを適用（動的生成を含む）

2. **差分確認モーダル**
   - 左右2カラムのレイアウト
   - 各カラム上部にリビジョン選択ドロップダウン
   - 2つの比較モード（Markdown / HTML）の切り替え

3. **リビジョン選択ドロップダウン**
   - 表示項目：`revisionNo - revisionId - 更新日時`
   - revisionNo：更新日時でソートし、古いものから1始まりの連番を割り当て

4. **Markdown比較モード**
   - 2つのリビジョンのMarkdownテキストを並べて差分表示
   - 差分箇所を視覚的にハイライト

5. **HTML比較モード**
   - GrowiのMarkdown→HTMLレンダリングエンジンを使用
   - 差分箇所を背景色でハイライト
     - 削除：淡いピンク（旧リビジョン側）
     - 追加：淡い緑（新リビジョン側）

## 成功の定義

- Growi Sidebar上のボタンから差分確認モーダルを開くことができる
- リビジョンを2つ選択してMarkdownおよびHTML形式の差分が正しく表示される
- Growi標準UIと統一感のあるデザインで提供される
- Growiプラグイン仕様に準拠して動作する

## ビジネス要件

- OSSプラグインとしてGrowiコミュニティに公開可能な品質
- Growiのプラグインシステム（growi-plugin）に準拠した実装

## ユーザーストーリー

1. **Wikiユーザーとして**、閲覧中の記事のSidebarにある差分確認ボタンをクリックすることで、差分確認モーダルを開きたい
2. **Wikiユーザーとして**、左右のドロップダウンから比較したい2つのリビジョンを選択し、差分を確認したい
3. **Wikiユーザーとして**、Markdown形式の生テキストで差分を確認したい
4. **Wikiユーザーとして**、HTMLレンダリング済みの見た目で差分を確認したい

## 受け入れ条件

- [ ] SidebarにGrowiネイティブUIと同一スタイルのボタンが表示される
- [ ] ボタンクリックでモーダルが開く
- [ ] モーダル内のドロップダウンに「revisionNo - revisionId - 更新日時」形式でリビジョン一覧が表示される
- [ ] revisionNoは更新日時ソート（古い順）で1始まりの連番が正しく割り当てられる
- [ ] Markdown比較モードで差分ハイライト表示が機能する
- [ ] HTML比較モードでGrowiレンダリングエンジンを用いた差分ハイライト表示が機能する
- [ ] HTML差分の削除箇所は淡いピンク、追加箇所は淡い緑でハイライトされる

## 機能要件

| ID | 機能 | 優先度 |
|---|---|---|
| F-01 | Sidebarボタン配置 | 高 |
| F-02 | 差分確認モーダル（左右レイアウト） | 高 |
| F-03 | リビジョン選択ドロップダウン | 高 |
| F-04 | revisionNo計算（日付ソート＋連番） | 高 |
| F-05 | Markdown差分比較モード | 高 |
| F-06 | HTML差分比較モード | 高 |
| F-07 | HTML差分ハイライト（ピンク/緑） | 高 |

## 非機能要件

- Growi既存UIとの一貫性（デザイン・スタイル）
- プラグインとして独立して動作（Growiコア改変不要）
- パフォーマンス：リビジョン取得・レンダリングが3秒以内に完了
- ブラウザ対応：Growiがサポートするモダンブラウザ
